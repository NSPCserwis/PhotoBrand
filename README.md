[PL]
⚠️ Ważna Uwaga dot. Antywirusów
Ponieważ plik ".exe" jest nowy i nie posiada płatnego certyfikatu Microsoft, 
Windows Defender lub programy antywirusowe mogą wyświetlić komunikat: "System Windows ochronił ten komputer". 
Użytkownik powinien wtedy kliknąć "Więcej informacji" -> "Uruchom mimo to". 
Jest to standardowa procedura dla autorskich programów.

[ENG]
⚠️ Important Note regarding Antiviruses
Because the ".exe" file is new and does not have a paid Microsoft certificate, 
Windows Defender or antivirus programs may display the message: "Windows has protected this computer." 
The user should then click "More Info" -> "Run Anyway". 
This is standard procedure for proprietary programs.
